# Django_Blog

## Minimal Starter

### install Python v3 >

### install virtual environment:
  
        python -m venv [name VE]
  
### start virtual environment:

  Linux:
  
        source [name VE]/bin/activate
        
  Win:
  
        [name VE]\Scripts\activate.bat
       
### open folder my_blog

### start server:

        python manager.py runserver
